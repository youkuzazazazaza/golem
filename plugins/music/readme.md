# Music 插件

音乐点播插件，通过关键词触发搜索并返回音乐卡片消息。

## 功能特性

- 关键词触发搜索歌曲
- 返回音乐卡片（AppMsg），支持播放、封面、歌词
- 随机 AppID：从 220+ 个开放平台 AppID 中随机选取，使来源显示更具多样性

## 使用方式

在聊天中发送以下任一前缀 + 歌曲名即可触发：

```
音乐 晴天
点歌 稻香
music bohemian rhapsody
```

插件会调用 QQ 音乐搜索接口，返回第一条匹配结果的音乐卡片，包含歌名、歌手、音频链接、专辑封面和歌词。

## 工作流程

1. 接收文本消息，检测是否以 `音乐 `、`点歌 `、`music ` 开头
2. 提取歌曲名，URL 编码后请求搜索 API
3. 解析 JSON 响应，取第一条结果
4. 构造 AppMsg XML（含随机 AppID）
5. 以音乐卡片消息发送给发送者

## 配置

无需额外配置，开箱即用。

## 文件结构

```
plugins/music/
├── main.go       # 插件主逻辑（元数据、事件订阅、OnEvent 处理）
├── provider.go   #  AppID 映射表及随机选取函数
└── go.mod        # 模块定义
```

## 依赖

- `github.com/sbgayhub/golem/sdk` — Golem 插件 SDK

## 注意事项

- 搜索 API 为第三方服务（`https://109a.cn/API/qqyy/api.php`），可用性不受控
- 返回的是第一条搜索结果，暂不支持选择特定条目
- 插件优先级为 10，匹配成功后不会继续传递给后续插件（`Next: false`）
